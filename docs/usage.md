# Usage

JobFit CV Studio is designed as a step-by-step application workflow.

## Step 1: Add CV

Paste the CV or resume text into the CV input area. For best results, include the summary, experience, skills, education, and project sections.

## Step 2: Add Job Post

Paste the job description. Include responsibilities, requirements, preferred qualifications, and any role-specific keywords.

## Step 3: Choose Preferences

Set:

- Target role
- Output language
- Preferred tone
- Detail level

Supported output languages are English, Turkish, and Dutch.

## Step 4: Generate Application Kit

Click `Generate application kit`. If an API key is configured and sample data is not forced, the app uses API mode. Otherwise, it uses deterministic Demo Mode.

## Step 5: Review And Download

Review the tabs:

- Match
- CV bullets
- Cover letter
- LinkedIn
- Interview
- Checklist
- Report

Use the report tab to download Markdown or plain text output.
